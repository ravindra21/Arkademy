# Soal 6c
- Npm Atau Yarn
- **Node JS 7 KEATAS**
- MYSQL(Saya pakai MariaDB)

# Menjalankan Aplikasi 6c
- import database.sql dari dir arkademy/6/6c/database.sql
- konfigurasikan password,username,dll di dir arkademy/6/6c/db.js
